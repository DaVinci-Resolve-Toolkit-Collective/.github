## ğŸ”§ Usage notes & best practices
- Back up any important projects before importing templates.  
- Copy `.drp` example projects to a local editable folder before editing.  
- Place `.cube` LUTs in Resolve LUT folder and reload LUTs via Preferences.  
- Test render/preset settings on short clip before full export.  
- If an installer is blocked by antivirus, verify the file hash with the source and follow vendor guidance.
